Acerca del proyecto

GalRetro es una aplicación web desarrollada como proyecto del bootcamp de desarrollo web de BIT, donde el principal objetivo mio es mostrar mi pasión por los juegos y las consolas retro. El objetivo principal tecnico de este proyecto fue poner en práctica las siguientes habilidades:

  • Implementar lógica en JavaScript para interactividad y manejo de datos.
  • Aplicar HTML, CSS y Bootstrap para un diseño responsivo y estructurado.
  • Desarrollar funcionalidades dinámicas y validar formularios.

El código fuente está disponible en GitHub y el sitio se publica con GitHub Pages.
